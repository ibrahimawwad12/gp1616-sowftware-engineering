HTML Elements

HTML consists of various elements that define the structure of a webpage.

  • Headings: Use <h1> to <h6> for headings.
  • Paragraphs: Use <p> for paragraphs.
  • Links: Use <a> for links. Example: Visit Example
  • Lists: Use <ul> for unordered lists and <ol> for ordered lists.
